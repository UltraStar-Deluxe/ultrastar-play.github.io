Player & Mic Profiles

Made with Unity

Built on a professional game engine and modern tools to provide the best experience for developers and users.

Open Source

UltraStar Play belongs to everyone. Get involved, add features, ideas, styles, or translations on GitHub.

Create your own songs

The integrated song editor allows to easily create a sing-along version of any song.

Manipulate notes via drag and drop, change lyrics, copy&paste notes, import MIDI files, add notes via mic recording or button tapping, and edit song properties such as the video offset or gap before the lyrics start.

Checkout the documentation for details.
